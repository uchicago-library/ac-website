1. Introduction

Attachment Converter is an open-source command-line tool for digital archivists. Given an email mailbox in MBOX format, it will go through and batch convert all the attachments in the mailbox into preservation formats, putting each one back into the email from which it originated. The user can then re-open the new mailbox in a mail user agent, such as Apple Mail, and browse through the mailbox the normal way.

2. GitHub

We are currently in the pre-alpha phase, gearing up to our first round of user testing. One the project is released, we will provide multiple out-of-the-box installation options. However, as of April 2023, the only way to install Attachment Converter is to build it from source and then copy the binary.

Our GitHub page can be found here:
